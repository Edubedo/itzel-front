## Acerca del Proyecto

**Itzel** es un sistema turnomático desarrollado para [CFE] que permite la gestión eficiente de turnos de los usuarios y tener un flujo mas optimo. El sistema facilita operaciones CRUD (Crear, Leer, Actualizar, Eliminar) para una administración completa del flujo de turnos.

### Características Principales

- **Gestión de Turnos**: Creación, modificación y eliminación de turnos
- **Administración de Personal**: Control completo del personal por turnos
- **Reportes y Estadísticas**: Informes detallados de operaciones
- **Control de Accesos**: Diferentes niveles de permisos de usuario
- **Interfaz Intuitiva**: Diseño user-friendly para fácil adopción
